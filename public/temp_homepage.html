Your resources for every-day JavaScripting

MooTools are JavaScript libraries designed for the intermediate to advanced JavaScript developer. They allows you to write powerful and flexible code with its elegant, well documented, and coherent API.

MooTools code it's extensively documented and has meaningful variable names: a joy to browse and a snap to understand.

MooTools libraries are released under the Open Source MIT license, which gives you the possibility to use them and modify them in every circumstance.
